# Understanding-HTTP

**Author:** Daniel Hasberger  
---

## Summary
The main focus of this tutorial is the basics of how the web works by focusing specifically on the **HTTP protocol**, which is the system that allows web browsers and web servers to communicate. Readers will learn what HTTP is, how requests and responses function, and what happens behind the scenes when visiting a website. The overall goal is to provide a simple and beginner friendly explanation of one of the most essential processes behind web browsing.
---

## Target Audience
This tutorial is intended for:
- Students with little to no understanding of how the web works 
- Readers with little or no prior networking experience  
- Anyone interested in learning how websites load and/or how browsers communicate with servers  
No technical background is required to be able to understand this tutorial.
