# Understanding-HTTP

**Author:** Daniel Hasberger  
---

## Summary
This tutorial introduces the basics of how the web works by focusing on the **HTTP protocol**, the system that allows web browsers and web servers to communicate. Readers will learn what HTTP is, how requests and responses function, and what happens behind the scenes when visiting a website. The goal is to provide a simple, beginner-friendly explanation of the most essential process behind modern web browsing.
---

## Target Audience
This tutorial is intended for:
- Students who are new to web technologies  
- Readers with little or no prior networking experience  
- Anyone interested in learning how websites load and how browsers communicate with servers  
- Ages **13+**, or any beginner-level learner  
No technical background is required.
